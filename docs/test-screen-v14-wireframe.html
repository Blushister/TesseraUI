Acceptance Criteria — TesseraUI v1.4

  • AC-01 · parser #text — text after a child element is emitted as a #text child node, not appended to node.text()
  • AC-02 · parser order — lead text, inline elements and tail text appear in DOM insertion order in node.children()
  • AC-03 · backward compat — plain <p>text</p> (no children) unchanged: text in node.text(), empty children list
  • AC-04 · hasInlineChildren — returns true when node has at least one #text or inline-tag child
  • AC-05 · collectRuns lead — node.text() (lead text) becomes first TextRun with parent style
  • AC-06 · collectRuns #text — #text children become TextRuns with parent style
  • AC-07 · collectRuns strong/b — <strong>/<b> produce TextRun with fontWeight=700
  • AC-08 · collectRuns em/i — <em>/<i> produce TextRun with color=CREAM_DIM (visual italic hint)
  • AC-09 · collectRuns span — <span style="color:…"> applies explicit CSS color to its TextRun
  • AC-10 · collectRuns nested — inline elements nested inside each other (e.g. <strong><span>…) merge styles correctly
  • AC-11 · text-decoration — underline and line-through on inline spans render decorations per token
  • AC-12 · word wrap — tokens that exceed the widget width start on the next line; a word wider than the widget renders on its own line
  • AC-13 · height auto — TesseraRichLabel.height equals the sum of all wrapped lines after reflow()
  • AC-14 · reflow on resize — setSize(w, h) triggers reflow() when width changes, updating reported height
  • AC-15 · <p> plain fallback — <p> with no inline children still produces a plain wrapped TesseraLabel
  • AC-16 · <h1>–<h6> — headings with inline children route to TesseraRichLabel with correct UA font-size and bold weight
  • AC-17 · <li> prefix — list items with inline children prepend bullet/number as a plain TextRun before the content runs
  • AC-18 · flattenText #text — updated flattenText() handles #text children correctly for the plain-label fallback path
  • AC-19 · #text in block — a #text child of a block element (not <p>/heading) renders as a wrapped TesseraLabel
  • AC-20 · /tessera test-v14 — command registered; opens TesseraTestScreenV14
