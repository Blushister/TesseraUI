V19 — Tabs · Modal · Focus

Commande in-game : /tessera test-v19

TesseraTabPanel

  • Tag HTML : <tabs> + <tab label="...">
  • Barre d'onglets auto-générée
  • Contenu switché sans rebuild

TesseraModal

  • Clique "Ouvrir modal" → overlay sombre + dialog
  • Touches Escape pour fermer
  • TesseraModal.open(panel) / .close()
  • Géré dans TesseraScreen.handleModalEvents()

TesseraFocusManager

  • Tab → focus suivant
  • Shift+Tab → focus précédent
  • Inputs enregistrés automatiquement sauf tabindex="-1"
  • TesseraFocusManager.register(widget)

TesseraScreen

  • beforeInit() → clear FocusManager
  • keyPressed() auto : Escape, Tab, I (debug)
