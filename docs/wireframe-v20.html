V20 — VirtualList · Transitions CSS

Commande in-game : /tessera test-v20

TesseraVirtualList

  • Tag HTML : <virtual-list row-height="16" v-for="...">
  • Seules les lignes visibles sont rendues
  • Cache LRU des widgets de lignes
  • Scroll natif avec clippage scissor
  • Slider contrôle la hauteur de ligne dynamiquement

CSS Transitions (TesseraPanel)

  • Propriété CSS transition: <property> <ms>
  • Hover les cartes → animation fluide
  • 4 propriétés supportées :
  • background — interpolation couleur de fond
  • opacity — fondu transparent
  • border-color — flash de bordure
  • color — changement couleur du texte enfant
  • all — active toutes les propriétés

Java API

  • TesseraVirtualList.of(items, rowH, factory)
  • panel.transition(300, "background")
  • panel.hoverColor(0xFFfbbf24)
