V21 — ContextMenu · Components · Debug

Commande in-game : /tessera test-v21

TesseraComponentRegistry

  • Définir un composant dans le HTML :
<template name="card">
  <col class="card">
    <row class="card-header">
      <slot name="title"/>
    </row>
    <slot/>
  </col>
</template>
  • Utilisation : <card><h3 slot="title">...</h3></card>
  • <slot> = contenu enfant injecté
  • <slot name="x"> = slot nommé

TesseraContextMenu

  • Clic droit sur une carte → menu contextuel
  • Attribut HTML : oncontextmenu="handler"
  • Builder Java : TesseraContextMenu.Builder
  • Rendu via renderTesseraOverlays()

TesseraDebugOverlay

  • Touche I → toggle overlay
  • Couleurs par profondeur :
  • Rouge — profondeur 0
  • Vert — profondeur 1
  • Bleu — profondeur 2
  • Jaune — profondeur 3+
