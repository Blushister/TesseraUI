V22 — Item Slots · Drag & Drop

Commande in-game : /tessera test-v22

TesseraItemSlot

  • Tag HTML : <item-slot size="24" item="minecraft:diamond"/>
  • Rendu natif d'un ItemStack Minecraft
  • Attribut show-count="false" → pas de chiffre
  • Slot vide = carré gris avec bordure

TesseraItemGrid

  • new TesseraItemGrid(cols, rows, slotSize)
  • Drag & drop entre slots automatique
  • .setItem(col, row, stack)
  • .getItem(col, row)

TesseraDragContext

  • État global du drag (un seul drag actif)
  • TesseraDragContext.startDrag(src, payload, x, y)
  • TesseraDragContext.isDragging()
  • TesseraDragContext.endDrag(x, y)
  • Ghost widget suit le curseur

TesseraDropZone

  • Interface : dropBounds() + accepts(payload) + onDrop(payload)
  • Wiring HTML : oncontextmenu="..."
  • Ou Java : panel.dropZone(handler)
  • Panel source : draggable="true" drag-payload="..."
